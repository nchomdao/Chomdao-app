# Chomdao — ระบบจัดการหอพัก

ระบบจัดการหอพักออนไลน์สำหรับ Chomdaoplace & Chomdaoresort

## ฟีเจอร์

- 🏠 จัดการห้องพัก — แยกหอพัก, แยกชั้น, ประเภทห้อง (แอร์/เฟอร์/ห้องเปล่า)
- 📋 บันทึกค่าใช้จ่าย — ค่าห้อง, น้ำ, ไฟ, ค่าอื่นๆ (จอดรถ, เน็ต ฯลฯ)
- ⚡ จดมิเตอร์น้ำ-ไฟ — กรอกรวดเดียวทุกห้อง, คำนวณอัตโนมัติ
- 🖨 พิมพ์บิล PDF — ครึ่ง A4, โลโก้, เลขมิเตอร์, จับคู่ 2 บิล/หน้า
- 📊 สรุปรายรับ — กราฟรายเดือน, เลือกดูย้อนหลังได้
- ☁ เชื่อม Google Sheets — sync อัตโนมัติ, ลบข้อมูลเก่า 12 เดือน

## วิธีใช้

### ใช้งานผ่าน GitHub Pages
เปิด `https://<username>.github.io/chomdao-app/`

### ใช้งาน local
ดาวน์โหลด `index.html` แล้วเปิดในเบราว์เซอร์
